    - `CellInstance` is a `#[repr(C)]` `bytemuck::Pod+Zeroable` struct with: `cell_pos: [u32; 2]` (col, row), `fg: [f32; 4]`, `bg: [f32; 4]`, `uv: [f32; 4]`, `atlas_kind: u32` (0=Mono, 1=Color, 2=Empty/bg-only), `selected: u32` (0 or 1; bit-packed but kept as u32 for WGSL ergonomics), `flags: u32` (bit 0 = inverse, bit 1 = bold, others reserved).
    - `CellPipeline` owns: vertex buffer (4-vertex quad; reused across all cells via per-instance indexing), index buffer (6 indices for 2 triangles), instance buffer (capacity = cols × rows × size_of(CellInstance), reallocated on grid resize), bind group (mono atlas view + color atlas view + linear sampler + uniform buffer with cell_size_px + viewport_size_px + selection_tint), wgpu::RenderPipeline (cell.wgsl).
    - `cell.wgsl` vertex shader: maps `cell_pos` and the per-vertex quad corner to a screen-space NDC position using `viewport_size_px` and `cell_size_px` uniforms. Outputs `frag_uv` (within atlas UV range), `frag_fg`, `frag_bg`, `frag_atlas_kind`, `frag_selected`.
    - `cell.wgsl` fragment shader: branch on `frag_atlas_kind`:
      - 0 (Mono): `let mono = textureSample(mono_atlas, samp, frag_uv).rgb; let glyph = vec4<f32>(frag_fg.rgb * mono, max(mono.r, max(mono.g, mono.b)));` — premultiplied. Then blend with bg: `out = mix(frag_bg, glyph, glyph.a);`
      - 1 (Color): `let color = textureSample(color_atlas, samp, frag_uv); out = mix(frag_bg, color, color.a);`
      - 2 (Empty): `out = frag_bg;`
      Then if `frag_selected == 1u`: `out.rgb = mix(out.rgb, selection_tint.rgb, selection_tint.a);` (selection_tint = uniform, alpha controls blend strength — Claude's discretion final value, e.g. `vec4(0.27, 0.48, 0.78, 0.40)` xterm-ish blue selection).
    - Color conversion (`color_to_rgba(c: Color, named_palette: &[Rgba; 256], default_fg: Rgba, default_bg: Rgba) -> [f32; 4]`):
      - `Color::Named(NamedColor::Foreground)` → `default_fg`; `NamedColor::Background` → `default_bg`; named ANSI colors (Black/Red/Green/...) → indices 0..7 / 8..15 of the standard 256-palette;
      - `Color::Spec(Rgb { r, g, b })` → `[r/255.0, g/255.0, b/255.0, 1.0]` (24-bit truecolor — D-44, RENDER-04 §"24-bit truecolor"; verified Phase 2 02-02-SUMMARY tests `sgr_truecolor.rs`);
      - `Color::Indexed(i)` → `named_palette[i]` (256-color path via xterm-256 default palette; Plan 03-04 lets the user override; Phase 3 ships the canonical xterm-256 lookup table).
    - The xterm-256 palette is a static `[[f32; 4]; 256]` table embedded in `compositor.rs` (16 ANSI + 216 6×6×6 cube + 24 grayscale ramp — well-known table; transcribe from xterm sources, attribute the source in a comment).
    - **Compositor::render** does (under one critical-section discipline):
      1. `let mut t = term.lock();`
      2. `let dims = t.dims(); let cursor = t.cursor(); let damage_kind = match t.damage() { TermDamage::Full => DamageKind::Full, TermDamage::Partial(iter) => DamageKind::Partial(iter.map(|b| (b.line, b.left, b.right)).collect::<Vec<_>>()) }; t.reset_damage();`
      3. Snapshot the cells needed: for `Full`, iterate the entire visible viewport (`for r in 0..dims.1 { for c in 0..dims.0 { grid[Point::new(Line(r as i32), Column(c as usize))].clone() } }`); for `Partial`, only iterate the (line, left..=right) ranges.
      4. `drop(t);` BEFORE any GPU work (D-11, Pitfall 2 in 03-RESEARCH.md).
      5. Convert each Cell to a CellInstance: skip cells with `flags.contains(Flags::WIDE_CHAR_SPACER)` (Pitfall 4 — they're emitted from the lead cell); for the lead cell, the spacer cell is implicit (the cell quad spans 1 cell physically; future plan can widen lead cell to 2 cells if visual width matches grapheme width — for v1 we draw the lead character in its own cell and leave the spacer cell as bg-only).
      6. Get atlas slot: `let glyph = font_stack.rasterize(cell.c)?; let slot = atlas.slot_for(&queue, GlyphKey { character: cell.c, dpr_bucket: 1 }, &glyph);`
      7. Write the dirty rows' slice of the instance buffer via `queue.write_buffer(&instance_buf, row_offset, cast_slice(&new_instances))`.
      8. Begin the surface render pass with bg-color clear → draw cells (instanced quad) → draw cursor (separate pipeline; Task 2).

    4. **Create `crates/vector-render/src/shaders/cell.wgsl`**:
       ```wgsl
       struct Uniforms {
           viewport_size_px: vec2<f32>,
           cell_size_px: vec2<f32>,
           selection_tint: vec4<f32>,
       }

       @group(0) @binding(0) var mono_atlas: texture_2d<f32>;
       @group(0) @binding(1) var color_atlas: texture_2d<f32>;
       @group(0) @binding(2) var samp: sampler;
       @group(0) @binding(3) var<uniform> u: Uniforms;

       struct VertexOutput {
           @builtin(position) clip_position: vec4<f32>,
           @location(0) frag_uv: vec2<f32>,
           @location(1) frag_fg: vec4<f32>,
           @location(2) frag_bg: vec4<f32>,
           @location(3) frag_atlas_kind: u32,
           @location(4) frag_selected: u32,
       }

       @vertex
       fn vs_main(
           @location(0) vertex_pos: vec2<f32>,    // unit quad corner: (0,0) (1,0) (0,1) (1,1)
           @location(1) cell_pos: vec2<u32>,      // instance: col, row
           @location(2) fg: vec4<f32>,
           @location(3) bg: vec4<f32>,
           @location(4) uv_rect: vec4<f32>,       // u0, v0, u1, v1
           @location(5) atlas_kind: u32,
           @location(6) selected: u32,
           @location(7) flags: u32,
       ) -> VertexOutput {
           let cell_px = vec2<f32>(f32(cell_pos.x), f32(cell_pos.y)) * u.cell_size_px;
           let pos_px = cell_px + vertex_pos * u.cell_size_px;
           // Map [0, viewport] -> [-1, 1] with y flipped (wgpu NDC y is up).
           let ndc = vec2<f32>(
               (pos_px.x / u.viewport_size_px.x) * 2.0 - 1.0,
               1.0 - (pos_px.y / u.viewport_size_px.y) * 2.0,
           );
           var out: VertexOutput;
           out.clip_position = vec4<f32>(ndc, 0.0, 1.0);
           out.frag_uv = mix(uv_rect.xy, uv_rect.zw, vertex_pos);
           // Inverse flag (bit 0): swap fg/bg
           if ((flags & 1u) != 0u) {
               out.frag_fg = bg;
               out.frag_bg = fg;
           } else {
               out.frag_fg = fg;
               out.frag_bg = bg;
           }
           out.frag_atlas_kind = atlas_kind;
           out.frag_selected = selected;
           return out;
       }

       @fragment
       fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
           var out: vec4<f32>;
           if (in.frag_atlas_kind == 0u) {
               // Mono: sample RGB alphamask, multiply by fg.
               let s = textureSample(mono_atlas, samp, in.frag_uv);
               let cov = max(s.r, max(s.g, s.b));
               let glyph = vec4<f32>(in.frag_fg.rgb * s.rgb, cov);
               out = vec4<f32>(mix(in.frag_bg.rgb, glyph.rgb, glyph.a), 1.0);
           } else if (in.frag_atlas_kind == 1u) {
               // Color: sample as-is.
               let s = textureSample(color_atlas, samp, in.frag_uv);
               out = vec4<f32>(mix(in.frag_bg.rgb, s.rgb, s.a), 1.0);
           } else {
               out = vec4<f32>(in.frag_bg.rgb, 1.0);
           }
           if (in.frag_selected == 1u) {
               out = vec4<f32>(mix(out.rgb, u.selection_tint.rgb, u.selection_tint.a), 1.0);
           }
           return out;
       }
       ```
