<objective>
Implement the pass-through proxy binary `vector-headless` per D-36:
1. Spawn `$SHELL` via `LocalDomain::spawn(SpawnCommand)` (Plan 02-04).
2. Bridge parent terminal stdin (in raw mode) to the PTY writer.
3. Pump PTY output through `vector_term::Term` (Plan 02-02) via a Term wrapped in a `parking_lot::Mutex` (lock-mutate-drop pattern; never held across await).
4. At ~30Hz, render the current `Term::grid()` back to the parent terminal as ANSI bytes (full repaint; dirty tracking is Phase 3's concern).
5. Watch parent SIGWINCH and propagate to both `Term::resize` AND `transport.resize` so the child reflows (CORE-04).
6. On child exit (signaled via oneshot from the transport-owning actor task — see §key_links), drain remaining PTY output, render final grid, exit 0.
7. Restore parent terminal raw mode via `scopeguard` on every exit path including panic.

**Transport ownership pattern (fixes held-mutex-across-await issue):** The `Box<dyn PtyTransport>` is owned by a single `transport_actor` task — NOT wrapped in `Arc<Mutex<…>>` and shared. Writers and SIGWINCH send commands via `mpsc::Sender<WriteCmd>` / `mpsc::Sender<ResizeCmd>`; the actor `select!`s over the two channels and serially calls `transport.write`/`transport.resize`. Child exit is detected by the reader-pump task observing `reader_rx.recv() == None` (channel closes when the spawn_blocking reader hits EOF), which signals a `tokio::sync::oneshot` to the main loop. `transport.wait()` is called by the actor task AFTER the write/resize channels close, to harvest the exit status. This removes all `transport.lock().await` calls across any other `.await`, eliminating the SIGWINCH-blocked-by-wait deadlock from the prior plan revision.

Per D-36 + VALIDATION.md, this plan's success is also gated by a manual checkpoint: the user runs vim, tmux, htop, and `less +F` inside vector-headless and confirms they render without visible corruption.

3. Create `crates/vector-headless/src/bridge.rs` — bridge tasks + transport_actor + shared Term holder:
   ```rust
   //! Bridge tasks (sole owners of their respective resources — no shared Mutex on transport):
   //!   - pump_pty_to_term:   owns reader_rx; pushes bytes into Term (parking_lot::Mutex, lock-mutate-drop).
   //!                         When reader_rx closes (child PTY EOF), signals exit via oneshot.
   //!   - pump_stdin_to_pty:  reads stdin on a spawn_blocking thread; forwards bytes to write_tx.
   //!   - transport_actor:    SOLE owner of Box<dyn PtyTransport>. select!s over write_rx + resize_rx;
   //!                         calls transport.wait() AFTER both channels close to harvest exit code.
   //!
   //! Critical invariant: NO `tokio::sync::Mutex` wraps the transport. Locks held across
   //! `.await` are forbidden by `clippy::await_holding_lock = "deny"` (D-11) AND would have
   //! caused a SIGWINCH-vs-wait deadlock if used here (see <objective> for the prior bug).

   use std::io::Read;
   use std::sync::Arc;

   use anyhow::Result;
   use parking_lot::Mutex;
   use tokio::sync::{mpsc, oneshot};
   use vector_mux::PtyTransport;
   use vector_term::Term;

   pub type SharedTerm = Arc<Mutex<Term>>;

   /// Resize command: (rows, cols, px_w, px_h) — matches PtyTransport::resize signature.
   pub type ResizeCmd = (u16, u16, u16, u16);

   /// Pumps PTY output through Term. Lock-mutate-drop on the Term Mutex (parking_lot;
   /// no `.await` inside the guard). When reader_rx returns None — meaning the child's
   /// PTY closed (Pitfall 7: spawn_blocking reader saw EOF) — signal main loop to exit.
   pub async fn pump_pty_to_term(
       mut reader: mpsc::Receiver<Vec<u8>>,
       term: SharedTerm,
       exit_signal_tx: oneshot::Sender<()>,
   ) {
       while let Some(chunk) = reader.recv().await {
           // Lock acquired and dropped within this scope; no `.await` inside the guard.
           let mut t = term.lock();
           t.feed(&chunk);
           drop(t);
       }
       tracing::info!("pty reader channel closed; signaling exit");
       let _ = exit_signal_tx.send(());
   }

   /// Pumps raw stdin bytes to write_tx. stdin is read on a blocking thread
   /// (spawn_blocking) because tokio's AsyncRead on stdin doesn't play with raw
   /// mode on macOS reliably. Drops write_tx (closing the channel) on EOF.
   pub async fn pump_stdin_to_pty(write_tx: mpsc::Sender<Vec<u8>>) -> Result<()> {
       let (chunk_tx, mut chunk_rx) = mpsc::channel::<Vec<u8>>(64);
       tokio::task::spawn_blocking(move || {
           let stdin = std::io::stdin();
           let mut stdin = stdin.lock();
           let mut buf = [0u8; 4096];
           loop {
               match stdin.read(&mut buf) {
                   Ok(0) => break, // EOF
                   Ok(n) => {
                       if chunk_tx.blocking_send(buf[..n].to_vec()).is_err() { break; }
                   }
                   Err(_) => break,
               }
           }
       });
       while let Some(bytes) = chunk_rx.recv().await {
           if write_tx.send(bytes).await.is_err() { break; }
       }
       Ok(()) // drops write_tx — transport_actor's write_rx will close
   }

   /// SOLE owner of the Box<dyn PtyTransport>. Receives write+resize commands
   /// via mpsc; on `else` branch (both channels closed) drains the wait future.
   /// Returns the final exit status via `done_tx`.
   pub async fn transport_actor(
       mut transport: Box<dyn PtyTransport>,
       mut write_rx: mpsc::Receiver<Vec<u8>>,
       mut resize_rx: mpsc::Receiver<ResizeCmd>,
       done_tx: oneshot::Sender<Result<Option<i32>>>,
   ) {
       loop {
           tokio::select! {
               biased;
               Some((rows, cols, pw, ph)) = resize_rx.recv() => {
                   if let Err(e) = transport.resize(rows, cols, pw, ph) {
                       tracing::warn!("transport.resize failed: {e}");
                   }
               }
               Some(bytes) = write_rx.recv() => {
                   if let Err(e) = transport.write(&bytes).await {
                       tracing::warn!("transport.write failed: {e}; closing writer side");
                       break;
                   }
               }
               else => break,
           }
       }
       // Both channels closed (or write errored) — harvest exit status.
       let status = transport.wait().await.map_err(|e| anyhow::anyhow!("transport.wait: {e}"));
       let _ = done_tx.send(status);
   }
   ```

   Notes for the executor:
   - `tokio::select! { biased; ... }` prioritizes resize over write so SIGWINCH is never starved by a hot write stream.
   - `transport.wait()` is called **after** the select loop terminates — never concurrently with write/resize. No held-lock-across-await.
   - The trait shape from Plan 02-04 is honored as-is — no surgery.

4. Replace `crates/vector-headless/src/main.rs` (the Plan 02-01 placeholder):
   ```rust
   //! vector-headless: pass-through terminal proxy (D-36).
   //!
   //! Architecture:
   //!   - winit-style threading does NOT apply here (no GUI).
   //!   - tokio multi-thread runtime owns: PTY bridge, stdin reader, SIGWINCH watcher,
   //!     transport_actor (sole owner of Box<dyn PtyTransport>), render tick.
   //!   - One `rt.block_on(...)` (allowlisted by tests/no_tokio_main.rs BLOCK_ON_ALLOWLIST).
   //!
   //! Public entry: `fn main() -> anyhow::Result<()>`.

   mod bridge;
   mod cli;
   mod render;
   mod sigwinch;

   use std::sync::Arc;
   use std::time::Duration;

   use anyhow::{Context, Result};
   use clap::Parser;
   use parking_lot::Mutex;
   use tokio::sync::{mpsc, oneshot};
   use tracing_subscriber::{fmt, EnvFilter};
   use vector_mux::{Domain, LocalDomain, SpawnCommand};
   use vector_term::Term;

   use crate::bridge::{
       pump_pty_to_term, pump_stdin_to_pty, transport_actor, ResizeCmd, SharedTerm,
   };

   fn parent_size_or(default_cols: u16, default_rows: u16) -> (u16, u16) {
       crossterm::terminal::size().map(|(c, r)| (c, r)).unwrap_or((default_cols, default_rows))
   }

   fn main() -> Result<()> {
       let cli = cli::Cli::parse();

       let filter = if cli.debug_parser {
           EnvFilter::new("vector_term=trace,vector_pty=trace,vector_mux=trace,vector_headless=trace")
       } else {
           EnvFilter::try_from_default_env().unwrap_or_else(|_| EnvFilter::new("warn"))
       };
       fmt().with_writer(std::io::stderr).with_env_filter(filter).init();

       let (default_cols, default_rows) = parent_size_or(80, 24);
       let cols = cli.cols.unwrap_or(default_cols);
       let rows = cli.rows.unwrap_or(default_rows);

       let rt = tokio::runtime::Builder::new_multi_thread()
           .enable_all()
           .thread_name("vector-headless-worker")
           .build()
           .context("build tokio runtime")?;

       // Single allowlisted block_on (BLOCK_ON_ALLOWLIST = &["main.rs"]).
       rt.block_on(run(cols, rows, cli.scrollback))
   }

   async fn run(cols: u16, rows: u16, scrollback: usize) -> Result<()> {
       // 1. Raw mode on parent terminal; ensure restoration on every exit path (Pitfall 4).
       crossterm::terminal::enable_raw_mode().context("enable_raw_mode")?;
       let _raw_guard = scopeguard::guard((), |_| {
           let _ = crossterm::terminal::disable_raw_mode();
           // Best-effort: clear screen + return cursor to top.
           print!("\x1b[2J\x1b[H");
           use std::io::Write;
           let _ = std::io::stdout().flush();
       });

       // 2. Construct LocalDomain + spawn a PtyTransport. Take the reader BEFORE
       //    handing transport to the actor — `take_reader` requires &mut self and
       //    can only be called from the owner.
       let domain = LocalDomain::new().context("LocalDomain::new (resolve shell)")?;
       let mut transport = domain
           .spawn(SpawnCommand { argv: None, cwd: None, rows, cols, env: vec![] })
           .await
           .context("Domain::spawn")?;
       let reader_rx = transport.take_reader().expect("take_reader first call");

       // 3. Shared Term (lock-mutate-drop on the parking_lot::Mutex, never across .await).
       let term: SharedTerm = Arc::new(Mutex::new(Term::new(cols, rows, scrollback)));

       // 4. Channels for the actor + exit signal.
       let (write_tx, write_rx) = mpsc::channel::<Vec<u8>>(64);
       let (resize_tx, resize_rx) = mpsc::channel::<ResizeCmd>(16);
       let (exit_signal_tx, exit_signal_rx) = oneshot::channel::<()>();
       let (done_tx, done_rx) = oneshot::channel::<Result<Option<i32>>>();

       // 5. transport_actor: sole owner of the transport. Drains write_rx + resize_rx;
       //    after both channels close, calls transport.wait() and reports via done_tx.
       let _actor = tokio::spawn(transport_actor(transport, write_rx, resize_rx, done_tx));

       // 6. stdin -> write_tx task.
       let _stdin_task = tokio::spawn(pump_stdin_to_pty(write_tx.clone()));

       // 7. PTY -> Term task; signals exit_signal_tx when reader_rx closes.
       let _pump_task = tokio::spawn(pump_pty_to_term(reader_rx, Arc::clone(&term), exit_signal_tx));

       // 8. SIGWINCH watcher: term.resize() inline + resize_tx.send() to actor.
       let _winch_task = tokio::spawn(sigwinch::watch(Arc::clone(&term), resize_tx.clone()));

       // 9. Render tick; exit when reader signals child EOF.
       let mut tick = tokio::time::interval(Duration::from_millis(33)); // ~30Hz
       let mut exit_signal_rx = exit_signal_rx;
       loop {
           tokio::select! {
               _ = tick.tick() => {
                   render::render_grid_to_stdout(&term)?;
               }
               _ = &mut exit_signal_rx => {
                   // Drain remaining output for ~100ms then render final.
                   tokio::time::sleep(Duration::from_millis(100)).await;
                   render::render_grid_to_stdout(&term)?;
                   break;
               }
           }
       }

       // 10. Drop senders so transport_actor's write_rx + resize_rx close — actor will
       //     then call transport.wait() and send via done_tx.
       drop(write_tx);
       drop(resize_tx);
       let exit_status = match done_rx.await {
           Ok(Ok(s)) => s,
           Ok(Err(e)) => { tracing::warn!("transport.wait error: {e}"); None }
           Err(_) => None,
       };
       tracing::info!(?exit_status, "child exited; vector-headless shutting down");
       Ok(())
   }
   ```

   ```rust
   use std::io::{stdout, Write};

   use anyhow::Result;

   use crate::bridge::SharedTerm;

   /// Render the current Term grid to parent stdout. Full repaint; ~12 KB per
   /// frame on 80x24 — acceptable for human-driver UX. Phase 3 will track dirty rows.
   pub fn render_grid_to_stdout(term: &SharedTerm) -> Result<()> {
       // Lock-mutate-drop: capture a snapshot of what we need, drop the lock, then write.
       // The grid borrow lives only within the lock guard; we copy cells into a local buffer.
       let frame = {
           let t = term.lock();
           build_frame(&t)
       };

       let mut out = stdout().lock();
       out.write_all(&frame)?;
       out.flush()?;
       Ok(())
   }

   fn build_frame(term: &vector_term::Term) -> Vec<u8> {
       let (cols, rows) = term.dims();
       let grid = term.grid();
       let (cur_col, cur_row) = term.cursor();
       let mut buf: Vec<u8> = Vec::with_capacity((cols as usize) * (rows as usize) * 16);

       // Cursor home + clear screen + hide cursor (avoids flicker on cursor relocation).
       buf.extend_from_slice(b"\x1b[?25l\x1b[H\x1b[2J");

       // Iterate visible rows; for each row, position cursor then emit cells.
       // NOTE: Grid indexing uses alacritty's Line + Column types. The exact
       // iteration pattern is `for row in 0..rows { for col in 0..cols { let cell = &grid[Point::new(Line(row), Column(col))]; ... } }`.
       // Plan 02-02 API surface exposes `grid()` — use its iteration helpers.
       for row in 0..rows as i32 {
           write!(&mut buf, "\x1b[{};1H", row + 1).ok();
           let mut last_fg: Option<u32> = None;
           let mut last_bg: Option<u32> = None;
           for col in 0..cols {
               let cell = cell_at(grid, row, col);
               let c = cell.c;
               // Emit color escapes lazily — only when fg/bg changes from prev cell.
               let (fg_code, bg_code) = cell_colors(&cell);
               if last_fg != Some(fg_code) {
                   write_fg(&mut buf, fg_code);
                   last_fg = Some(fg_code);
               }
               if last_bg != Some(bg_code) {
                   write_bg(&mut buf, bg_code);
                   last_bg = Some(bg_code);
               }
               // Emit char as UTF-8 bytes. Replace control chars with space to avoid corruption.
               if c.is_control() {
                   buf.push(b' ');
               } else {
                   let mut tmp = [0u8; 4];
                   buf.extend_from_slice(c.encode_utf8(&mut tmp).as_bytes());
               }
           }
           buf.extend_from_slice(b"\x1b[0m"); // reset SGR at end of row
       }

       // Reposition cursor where Term says it is + unhide.
       write!(&mut buf, "\x1b[{};{}H\x1b[?25h", cur_row + 1, cur_col + 1).ok();
       buf
   }

   // Helper: index a Grid cell at (row, col). Implementation depends on alacritty_terminal 0.26.
   // Pattern from Plan 02-02 conformance tests: `grid[(col, row).into()]` or `grid[Point::new(Line(row), Column(col))]`.
   // Use whichever Plan 02-02's tests confirm works.
   fn cell_at<'a>(
       grid: &'a alacritty_terminal::grid::Grid<alacritty_terminal::term::cell::Cell>,
       row: i32,
       col: u16,
   ) -> alacritty_terminal::term::cell::Cell {
       use alacritty_terminal::index::{Column, Line, Point};
       grid[Point::new(Line(row), Column(col.into()))].clone()
   }

   fn cell_colors(cell: &alacritty_terminal::term::cell::Cell) -> (u32, u32) {
       // Map cell.fg and cell.bg to encoded color codes. Format:
       //   0x01_RR_GG_BB for RGB (high byte = 1)
       //   0x00_00_00_NN for indexed (high byte = 0, low byte = palette index)
       //   0xFF_FF_FF_FF for "default" (sentinel)
       (encode_color(&cell.fg), encode_color(&cell.bg))
   }

   fn encode_color(c: &alacritty_terminal::vte::ansi::Color) -> u32 {
       // Variant names per Plan 02-01 API-SPIKE.md.
       // Adjust if spike says `Color::Rgb(Rgb)` instead of `Color::Spec(Rgb)`.
       use alacritty_terminal::vte::ansi::{Color, NamedColor};
       match c {
           Color::Named(NamedColor::Foreground) | Color::Named(NamedColor::Background) => 0xFFFFFFFF,
           Color::Named(n) => 0x0000_0000 | (*n as u32 & 0xFF),
           Color::Indexed(i) => 0x0000_0000 | (*i as u32 & 0xFF),
           Color::Spec(rgb) => 0x0100_0000 | ((rgb.r as u32) << 16) | ((rgb.g as u32) << 8) | (rgb.b as u32),
       }
   }

   fn write_fg(buf: &mut Vec<u8>, code: u32) {
       use std::io::Write;
       if code == 0xFFFFFFFF { buf.extend_from_slice(b"\x1b[39m"); return; }
       if code & 0x0100_0000 != 0 {
           let r = (code >> 16) & 0xFF;
           let g = (code >> 8) & 0xFF;
           let b = code & 0xFF;
           write!(buf, "\x1b[38;2;{};{};{}m", r, g, b).ok();
       } else {
           write!(buf, "\x1b[38;5;{}m", code & 0xFF).ok();
       }
   }

   fn write_bg(buf: &mut Vec<u8>, code: u32) {
       use std::io::Write;
       if code == 0xFFFFFFFF { buf.extend_from_slice(b"\x1b[49m"); return; }
       if code & 0x0100_0000 != 0 {

       } else {
           write!(buf, "\x1b[48;5;{}m", code & 0xFF).ok();
       }
   }
   ```

   ```rust
   //! Parent SIGWINCH watcher. Propagates to Term::resize (parking_lot lock-mutate-drop)
   //! AND sends a resize command to the transport_actor via mpsc (no held lock across await).
   //! transport_actor calls transport.resize() serially with write — the kernel delivers
   //! SIGWINCH to the foreground pgrp (CORE-04).

   use tokio::signal::unix::{signal, SignalKind};
   use tokio::sync::mpsc;

   use crate::bridge::{ResizeCmd, SharedTerm};

   pub async fn watch(term: SharedTerm, resize_tx: mpsc::Sender<ResizeCmd>) {
       let mut winch = match signal(SignalKind::window_change()) {
           Ok(s) => s,
           Err(e) => {
               tracing::warn!("cannot install SIGWINCH handler: {e}");
               return;
           }
       };
       while winch.recv().await.is_some() {
           let (cols, rows) = match crossterm::terminal::size() {
               Ok((c, r)) => (c, r),
               Err(e) => {
                   tracing::warn!("crossterm::terminal::size failed: {e}");
                   continue;
               }
           };
           tracing::debug!(cols, rows, "parent SIGWINCH; resizing term + transport");
           {
               // parking_lot::Mutex; lock-mutate-drop, no .await inside guard.
               let mut t = term.lock();
               t.resize(cols, rows);
               drop(t);
           }
           // Send to transport_actor — it owns the transport and calls resize serially.
           if resize_tx.send((rows, cols, 0, 0)).await.is_err() {
               tracing::info!("transport_actor closed resize_rx; sigwinch watcher exiting");
               return;
           }
       }
   }
   ```
