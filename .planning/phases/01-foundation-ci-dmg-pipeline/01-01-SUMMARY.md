## Decisions Made

- **Workspace `version = "2026.5.10"` instead of plan-literal `2026.05.10`.** Cargo's SemVer parser rejects leading-zero minor versions (`05`). CalVer date intent (2026-05-10) preserved. Downstream consumers of `pkg.version` see `2026.5.10`. Note in plan acceptance criteria that referenced `"2026.05.10"` is superseded by this commit; the contract (single workspace version + CalVer date) is intact.
- **vector-app ships both `main.rs` and `lib.rs` at this stage.** Per the plan, the `[[bin]]` section explicitly points at `src/main.rs`; the placeholder `lib.rs` exists for the future unit-test seam that Plan 01-03 will exercise.
- **`_force_anyhow_use` shim used instead of `[lints.rust.unused_imports = "allow"]`.** The shim is local to each stub crate, removable file-by-file as real code lands, and does not require a workspace-wide lint policy that Plan 01-02 will set authoritatively.
